1. Create a database and add its detail in the code:

create table users (id int primary key auto_increment not null, username varchar(100), password varchar(100), role varchar(100));


2. Add rows in tables:
e.g.
insert into users values (1, 'tom', '1234', 'admin');
insert into users values (2, 'brad', '12345', 'user');
insert into users values (3, 'matt', 'pass1234', 'user');



3. To run type this command on terminal:

venv\scripts\activate
python .\app.py
